This directory contains the general modules what doesn't belongs to any existing namespace

Initialize in html:

    <!-- utilities js START -->
    <script src="../core/utilities/utils.js"></script>
    <script src="../core/utilities/errorHandler.js"></script>
    <script src="../core/utilities/settings.js"></script>
    <script src="../core/utilities/message.js"></script>
    <script src="../core/utilities/googleAnalitics.js"></script>


Initialize in manifest content scripts:

    /* utilities js START*/
    "core/utilities/utils.js",
    "core/utilities/errorHandler.js",
    "core/utilities/settings.js",
    "core/utilities/message.js",
    "core/utilities/googleAnalitics.js",
